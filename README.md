# Buscador de Frases - Frontend e Backend

Este repositório contém o código-fonte de um projeto desenvolvido para criar uma API que busca frases do site "O Pensador" e uma página frontend para consumir e testar essa API.

## Frontend

O frontend consiste em uma página HTML com um buscador de frases. Os resultados são exibidos na página conforme a busca realizada pelo usuário.

### Estrutura do Projeto

- **HTML:** Página principal com um campo de busca e botão para encontrar uma nova frase.
- **CSS:** Estilização simples para melhorar a aparência da página.
- **JavaScript:** Lógica para fazer solicitações à API e exibir as frases na página.

### Como Executar

1. Abra o arquivo `index.html` em um navegador web.
2. Insira um termo de pesquisa no campo e clique no botão "Nova Frase" para obter uma frase do Pensador.

### Referência

Neste projeto, uma API foi desenvolvida para buscar frases do site "O Pensador". As frases são recuperadas de forma aleatória. Caso deseje um tema específico, digite o assunto e busque a nova frase. Visite o site [O Pensador](https://www.pensador.com/) – uma ótima fonte de frases, poemas, textos, entre outros.

## Backend

O backend é uma API construída em Node.js usando o framework Micro. Ela realiza scraping na página de busca do Pensador para obter frases relacionadas ao termo pesquisado.
